# Portfolio Assets

This folder contains assets for showcasing the AWS Quiz Application development story.

## Structure

- `infographic-story/` - Content and assets for the NotebookLM-generated presentation
- `demo-content/` - Sample questions and content for demo mode
- `chatbot-knowledge/` - Knowledge base content for the AI chatbot
- `screenshots/` - Application screenshots for documentation
- `diagrams/` - Architecture and flow diagrams

## Usage

These assets support the portfolio enhancement features that demonstrate:
1. Full-stack development capabilities
2. AWS serverless architecture expertise  
3. AI integration and transformation skills
4. User experience and design thinking
